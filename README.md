# Задание 1: Система учета времени выполнения методов

Приложение содержит:
Аннотации:
@TrackTime  - для синхронного отслеживания времени выполнения методов
@TrackAsyncTime - - для асинхронного отслеживания времени выполнения методов


Аспекты:

      TrackTimeAspect - определение времени выполнения метода, помеченного аннотацией @TrackTime и записи результата в БД.

            Данной аннотацией помечен сервис  GET /api/animal/metricTypes  Выборка типов животных

      TrackAsyncTimeAspect - асинхронное определение времени выполнения метода, помеченного аннотацией @TrackAsyncTime и записи результата в БД.

            Данной аннотацией помечен сервис  GET /api/animal/animals Выборка животных

      CheckAccessAspect - проверка доступа к заданному методу. 

            Данной аннотацией помечен сервис  PATCH  /api/animal/animals/{id} Изменение животного

REST API

       REST API разработан с использованием принципа API First  на основе спецификации Open API и плагина генерации кода на основе спецификации.

       При локальном запуске список API и структуру данных можно посмотреть по адресу:   http://localhost:8081/swagger-ui/index.html#/ 

       Rest сервис GET /api/admin/trackTimeMetrics предназначен для дыборка статистики выполнения методов   



Вход в систему

      Для входа в систему нужно вызвать POST /api/login  Вход в систему и указать 
      {
        "username": "ADMIN",
               "password": "ADMIN"
             }

или

     {
              "username": "developer",
             "password": "developer"
          }

      Два пользователя с разными используются для демонстрации использования АОП для управления доступом к методам.

База данных

      В приложении используется облачная БД из VK Cloud.  Настройка осуществлена в файле property.
